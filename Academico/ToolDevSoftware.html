HERRAMIENTAS DE DESARROLLO DE SOFTWARE

Estas son las Herramientas para el Desarrollo de Software


HTML5 es la quinta y última versión principal del lenguaje de marcado de hipertexto (HTML), que se utiliza para estructurar y presentar documentos en la World Wide Web. Es un estándar que define la estructura, el diseño y el contenido de una página web, incluyendo elementos como texto, imágenes, videos, y más. CSS3 (Cascading Style Sheets 3) es la última versión del lenguaje CSS, un lenguaje de diseño web que define la presentación visual de documentos HTML y XML, manteniendo separados el contenido del diseño. Permite a los diseñadores crear interfaces modernas, interactivas y responsivas. JavaScript (JS) es un lenguaje de programación que permite agregar interacciones y dinamismo a páginas web, además de ser utilizado en diversas aplicaciones y entornos. Es un lenguaje interpretado, lo que significa que el código se ejecuta línea por línea en un navegador web. Visual Studio Code (VS Code) es un editor de código fuente ligero y potente, gratuito y multiplataforma, desarrollado por Microsoft. Es ampliamente utilizado por desarrolladores para editar, depurar y compilar código.
HTML5 CSS3 JavaScript Visual Studio Code
GitHub es una plataforma online que permite alojar, compartir y colaborar en proyectos de código. Funciona como un "repositorio" donde se guarda el código fuente de un proyecto, utilizando el sistema de control de versiones Git. Ayuda a los desarrolladores a organizar y seguir las modificaciones en el código a lo largo del tiempo, facilitando la colaboración en equipo. Python es un lenguaje de programación interpretado de alto nivel, versátil y de código abierto.Python se utiliza en una amplia gama de aplicaciones, desde el desarrollo web y la ciencia de datos hasta la inteligencia artificial y la automatización. Java es un lenguaje de programación de tipo orientado a objetos, multiplataforma y de propósito general, ampliamente utilizado para desarrollar una amplia variedad de aplicaciones, desde aplicaciones web y móviles hasta software empresarial y aplicaciones de escritorio. Microsoft Azure es una plataforma de computación en la nube que ofrece una amplia gama de servicios, incluyendo computación, análisis, almacenamiento, redes y inteligencia artificial. Permite a las empresas crear, ejecutar y administrar aplicaciones en la nube, tanto en el entorno local como en el perímetro, utilizando las herramientas y marcos que prefieran.
